## How to Run This Project

Create the **.env** files for each project. You can duplicate the **.example.env** files and rename them to **.env**.

In the **backend** run:

- `yarn install`
- `yarn prisma dev` (\# Keep this running)
- `yarn build`
- `yarn start` (\# Keep this running)
- `yarn db:seed`

In the **frontend** run:

- `yarn install`
- `yarn build`
- `yarn start` (\# Keep this running)
